# Data Preprocessing Pipeline

## Overview

```
Raw ERA5 .nc (0.25deg)     Raw CONUS404 .nc (4km WRF)
        |                           |
        v                           v
  [xESMF regrid]             [direct load]
        |                           |
        v                           v
  ERA5 on CONUS grid         CONUS404 fields
  (1015x1367)                (1015x1367)
        |                           |
        +--- static fields ---+     |
        |                     |     |
        v                     v     v
  cached_data/era5_{year}.npy    cached_data/conus_{year}.npy
  cached_data/static_fields.npy
```

## ERA5 Data Format

- **Source:** ECMWF Reanalysis v5, 0.25-degree resolution
- **Files:** `data/era5_{year}.nc` (symlinks), years 1980-2020
- **Grid:** Regular lat/lon, 111 x 235 points
- **Time structure:** `(12, 366, 111, 235)` — 12 months x 366 days
  - Each month only populates its own days; other days are NaN
  - Requires month lookup: `_month_for_day(day_of_year)` to find which month index contains valid data

## CONUS404 Data Format

- **Source:** WRF dynamical downscaling of ERA5, 4km resolution
- **Files:** `data/conus404_yearly_{year}.nc` (symlinks), years 1980-2020
- **Grid:** WRF Lambert conformal projection, 1015 x 1367 points
- **Time:** `(366, 1015, 1367)` per variable per year

## Regridding (src/preprocessing/regrid.py)

ERA5 is on a regular lat/lon grid; CONUS404 is on a WRF Lambert conformal grid. These grids do not align.

**Method:** xESMF bilinear interpolation with nearest-neighbor extrapolation for boundary pixels.

- Weight matrix computed once at startup and held in memory
- ~32% of CONUS404 pixels fall outside ERA5 lat/lon bounds — handled by extrapolation
- The regridder maps ERA5 (111x235) -> CONUS404 grid (1015x1367)

## Static Fields (6 channels)

All pre-computed and z-scored during cache generation:

| Channel | Field | Source | Notes |
|---------|-------|--------|-------|
| 0 | Terrain height | CONUS404 Z variable (level 0) | Meters, z-scored |
| 1 | Orographic variance | Computed from terrain | Variance in 16x16 windows |
| 2 | Latitude | CONUS404 lat array | Normalized to [-1, 1] then z-scored |
| 3 | Longitude | CONUS404 lon array | Normalized to [-1, 1] then z-scored |
| 4 | LAI | ERA5 lai variable, regridded | NaN filled with 0.0, z-scored |
| 5 | Land-sea mask | ERA5 lsm, regridded + thresholded | Binary (0/1), z-scored |

Saved as: `cached_data/static_fields.npy` shape `(6, 1015, 1367)`

## Land Masking (src/preprocessing/land_mask.py)

- ERA5 land-sea mask regridded to CONUS404 grid
- Restricted to CONUS geographic bounds: 24-50N, 125-66W
- Pixels outside bounds or over ocean are excluded

**Patch filtering:**
- 256x256 patches extracted from 1015x1367 grid
- Only patches with >= 80% land fraction are used (`min_land_frac=0.8`)
- Result: 87 valid patch origins
- Non-land pixels within valid patches are filled with per-channel patch mean

## Normalization (src/preprocessing/normalization.py)

### Pretransforms (applied before z-scoring)

| Variable | Transform | Reason |
|----------|-----------|--------|
| tp, PREC_ACC_NC | log1p | Heavy right-skewed distribution |
| Q2 | sqrt | Right-skewed, non-negative |
| z | divide by g=9.81 | Convert geopotential to height |
| All others | none | Approximately Gaussian |

### Z-scoring

Statistics computed from training years (1980-2014), sampled from first 5 years every 30 days:
```
normalized = (pretransform(x) - mean) / std
```

Stored in `norm_stats.npz` for reuse across runs.

**Important:** Only ERA5 dynamic variable channels are normalized. Static fields are already z-scored during cache generation.

## NaN Day Handling

63 days across 4 years have fully-NaN CONUS404 data (verified in raw source .nc files):
- 2004: 28 days
- 2005: 32 days
- 2008: 1 day
- 2009: 2 days

Pre-computed in `cached_data/nan_days.json`. The dataset skips these days during training/evaluation.

## Cache Format

Per-year numpy files for fast mmap-based loading:

```
cached_data/
  era5_{year}.npy       # (N_days, N_era5_vars, 1015, 1367) float32
  conus_{year}.npy      # (N_days, N_conus_vars, 1015, 1367) float32
  static_fields.npy     # (6, 1015, 1367) float32
  nan_days.json         # {year_str: [day_indices]}
```

Generated by: `python -m src.preprocessing.cache_builder --all`
Parallelized via SLURM array job: `scripts/run_preprocess.sh`

## Verification

Run verification scripts to validate preprocessing:
```bash
python scripts/verify_preprocessing.py --plot    # cache vs raw .nc
python scripts/verify_normalization.py            # norm stats correctness
python -m src.preprocessing.verify_cache          # integrity + correlations
```
